  • Terminal: dotnet run

  3?? PROBAR EN SWAGGER
  ????????????????????????????????????????????????????

  • Prueba GET /api/products
  • Prueba GET /api/categories

  ????????????????????????????????????????????????????
  • Copia las interfaces de: Frontend/typescript-interfaces.ts
  • Actualiza tu servicio Angular
  • Cambia la URL base a: https://localhost:5006/api
